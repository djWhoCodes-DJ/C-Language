## Unformatted Output functions
- putchar()
- putch()
- puts()

---

### putchar()
- Defined inside stdio.h library
- char ch = 'a';
- putchar(ch); | o/p : a
- Cannot add string inside putchar, only character
- putchar('\n')
- putchar('D');

---

### putch()
- Defined inside conio.h library
- putch(ch);
- putch('\n);
- putch('D');

---

### puts()
- Slash or new line character is default in puts()
- puts("Devanshu Jain")
- string ch = "Devanshu"
- puts(ch)
- puts('a') | Error
